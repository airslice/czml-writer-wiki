#VerticalOrigin (value)

A vertical origin, which specifies whether an element is bottom-, center-, or top-aligned with respect to an origin.  Valid values are "BOTTOM", "CENTER", and "TOP".

**Type**: string

##Values

* `BOTTOM`

* `CENTER`

* `TOP`
