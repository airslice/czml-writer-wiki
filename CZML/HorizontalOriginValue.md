#HorizontalOrigin (value)

A horizontal origin, which specifies whether an element is left-, center-, or right-aligned with respect to an origin.  Valid values are "LEFT", "CENTER", and "RIGHT".

**Type**: string

##Values

* `LEFT`

* `CENTER`

* `RIGHT`
